🗄️ DBeaver y Bases de Datos

📚 ¿Qué es DBeaver?

DBeaver es una herramienta de software gratuito que permite conectarse a bases de datos, ejecutar consultas SQL, visualizar datos y gestionar las estructuras de las bases de datos. Es muy utilizada por desarrolladores y administradores de bases de datos.

💾 Conceptos de Bases de Datos

  • Base de Datos: Conjunto organizado de datos relacionados
  • Tabla: Estructura con filas (registros) y columnas (campos)
  • Registro: Una fila de datos en una tabla
  • Campo/Columna: Atributo de los datos (ej: nombre, edad)
  • Clave Primaria: Identificador único de cada registro
  • Índice: Acelera búsquedas en la base de datos

🔤 SQL (Structured Query Language)

SQL es el lenguaje estándar para interactuar con bases de datos. Permite crear, leer, actualizar y eliminar datos.

Crear Tabla

CREATE TABLE usuarios (
    id INTEGER PRIMARY KEY AUTO_INCREMENT,
    nombre VARCHAR(100) NOT NULL,
    email VARCHAR(100) UNIQUE,
    edad INTEGER,
    fecha_registro TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

Insertar Datos

INSERT INTO usuarios (nombre, email, edad) 
VALUES ('Juan', 'juan@email.com', 28);

INSERT INTO usuarios (nombre, email, edad) 
VALUES 
('María', 'maria@email.com', 25),
('Carlos', 'carlos@email.com', 30);

Consultar Datos

SELECT * FROM usuarios;                    # Todos los datos
SELECT nombre, email FROM usuarios;        # Campos específicos
SELECT * FROM usuarios WHERE edad > 25;   # Con condición
SELECT * FROM usuarios ORDER BY edad DESC; # Ordenado
SELECT * FROM usuarios LIMIT 10;           # Límite de resultados

Actualizar Datos

UPDATE usuarios SET edad = 26 WHERE id = 1;
UPDATE usuarios SET email = 'nuevo@email.com' WHERE nombre = 'Juan';

Eliminar Datos

DELETE FROM usuarios WHERE id = 1;
DELETE FROM usuarios WHERE edad < 20;

Consultas Avanzadas

# Contar registros
SELECT COUNT(*) FROM usuarios;

# Filtrar duplicados
SELECT DISTINCT edad FROM usuarios;

# Búsqueda con LIKE
SELECT * FROM usuarios WHERE nombre LIKE 'J%';

# Agrupar y contar
SELECT edad, COUNT(*) FROM usuarios GROUP BY edad;

# Unir tablas
SELECT u.nombre, p.titulo 
FROM usuarios u 
JOIN posts p ON u.id = p.usuario_id;

🖥️ Tipos de Datos SQL

  • VARCHAR(n): Texto de hasta n caracteres
  • INTEGER: Números enteros
  • FLOAT/DECIMAL: Números decimales
  • DATE: Fechas (YYYY-MM-DD)
  • TIMESTAMP: Fecha y hora
  • BOOLEAN: Verdadero o Falso
  • TEXT: Texto largo

⚙️ Usando DBeaver

1. Crear nueva conexión
2. Seleccionar tipo de BD (MySQL, PostgreSQL, etc)
3. Ingresar host, usuario, contraseña
4. Conectar
5. Crear o abrir base de datos
6. Ejecutar queries en el editor SQL
7. Ver resultados en tiempo real
