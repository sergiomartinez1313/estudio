🐧 Ubuntu y Terminal

📚 ¿Qué es Ubuntu?

Ubuntu es un sistema operativo basado en Linux, gratuito y de código abierto. Es muy utilizado por desarrolladores y profesionales de IT. Ubuntu te permite trabajar desde la terminal, ejecutar comandos poderosos y tener control total sobre tu sistema.

🎯 Comandos Básicos de Terminal

Navegación

pwd              # Muestra el directorio actual
ls               # Lista archivos y carpetas
cd carpeta       # Cambia de directorio
cd ..            # Sube un nivel
cd ~             # Va al home

Crear y Eliminar

mkdir carpeta              # Crea una carpeta
touch archivo.txt          # Crea un archivo vacío
rm archivo.txt             # Elimina un archivo
rm -r carpeta              # Elimina carpeta y contenido
cp archivo.txt copia.txt   # Copia un archivo
mv archivo.txt nuevo.txt   # Mueve o renombra

Ver Contenido

cat archivo.txt            # Muestra el contenido
less archivo.txt           # Muestra con scroll
head archivo.txt           # Primeras líneas
tail archivo.txt           # Últimas líneas

Permisos

chmod +x script.sh         # Hace ejecutable
chmod 755 archivo          # Permisos específicos
sudo comando               # Ejecuta como administrador

Búsqueda

find . -name "*.py"        # Busca archivos
grep "texto" archivo.txt   # Busca texto
whereis comando            # Ubica un comando

Otros Útiles

clear                      # Limpia la terminal
history                    # Muestra historial
whoami                     # Muestra usuario actual
date                       # Muestra fecha/hora
df -h                      # Espacio en disco
ps aux                     # Procesos activos

📦 Gestor de Paquetes - APT

APT es el gestor de paquetes de Ubuntu. Te permite instalar, actualizar y desinstalar programas fácilmente.

sudo apt update            # Actualiza lista de paquetes
sudo apt upgrade           # Actualiza programas
sudo apt install git       # Instala un programa
sudo apt remove git        # Desinstala
sudo apt search python     # Busca paquetes

💻 Git en Terminal

git init                   # Inicia repositorio
git status                 # Estado del repo
git add .                  # Prepara cambios
git commit -m "mensaje"    # Confirma cambios
git push                   # Sube cambios
git pull                   # Trae cambios
git clone URL              # Clona repositorio
