PASO 1: Abrir Portal

   Haz clic: "Intentar Desafío →"
   └─ Redirige a: editor.html?challenge=[ID]
   └─ Se carga el desafío en el editor

PASO 3: Ver Enunciado
   Panel izquierdo muestra:
   ✓ Título del desafío
   ✓ Dificultad (color-coded)
   ✓ Área temática
   ✓ XP a ganar (ej: ⭐ 75 XP)
   ✓ Enunciado completo con requisitos
   ✓ Hints y ejemplos de código

PASO 4: Escribir Código
   Editor central:
   ✓ Espacio para escribir tu código
   ✓ Syntax highlighting (monoespacio)
   ✓ Preview en tiempo real a la derecha

PASO 5: Completar
   Botón "✅ Terminar Prueba"
   ├─ Valida que hayas intentado (mensaje)
   ├─ Registra desafío como completado
   ├─ Suma XP (50/75/100)
   ├─ Verifica logros a desbloquear
   ├─ Muestra notificación de éxito
   └─ Vuelves a estudio.html

PASO 6: Ver Progreso
   En estudio.html:
   ✓ Desafío marca como "✅ Completado"
   ✓ XP subió (visible en barra)
   ✓ Level subió (si completó 100 XP)
   ✓ Badge desbloqueado (si se cumplió criterio)
