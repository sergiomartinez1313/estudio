1. Introducción a React

¿Qué es React?

React es una librería desarrollada por Facebook que permite crear interfaces dinámicas y responsivas. Utiliza componentes reutilizables y una forma eficiente de actualizar el DOM.

Características principales:

  • Componentes: Bloques reutilizables de UI
  • JSX: Sintaxis que mezcla JavaScript y HTML
  • Virtual DOM: Optimización de renderizado
  • Unidireccional: Flujo de datos de arriba hacia abajo
  • Hooks: Funcionalidades en componentes funcionales

Instalación y setup:

# Con Create React App (CRA)
npx create-react-app mi-proyecto
cd mi-proyecto
npm start

# Con Vite (más rápido)
npm create vite@latest mi-proyecto -- --template react
cd mi-proyecto
npm install
npm run dev

userEffect con Dependencias

// Sin dependencias: se ejecuta cada render
useEffect(() => {
  console.log('Cada render');
});

// Array vacío: se ejecuta solo al montar
useEffect(() => {
  console.log('Solo al montar');
}, []);

// Con dependencias: se ejecuta si las dependencias cambian
useEffect(() => {
  console.log('Cambió contador');
}, [contador]);

Cleanup en useEffect

Limpiar efectos cuando el componente se desmonta.

function Temporizador() {
  const [segundos, setSegundos] = useState(0);

  useEffect(() => {
    const intervalo = setInterval(() => {
      setSegundos(s => s + 1);
    }, 1000);

    // Cleanup: se ejecuta al desmontar
    return () => clearInterval(intervalo);
  }, []);

  return 
Segundos: {segundos}
;
}

Buenas Prácticas

  • Mantener componentes pequeños y enfocados
  • Usar hooks en lugar de clases cuando sea posible
  • Memoizar componentes cuando sea necesario (React.memo, useMemo)
  • Evitar crear funciones en el render (usar useCallback)
  • Separar lógica en custom hooks
  • Usar TypeScript para mejor tipado
  • Testing: Jest + React Testing Library
  • Performance: Code splitting con lazy y Suspense
