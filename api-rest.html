🔗 API REST - Arquitectura Web Profesional

REST (Representational State Transfer) es un estilo arquitectónico para diseñar APIs web. Las APIs REST son la forma estándar de comunicación entre aplicaciones en internet. Son, escalables y fáciles de entender.

🎯 Conceptos de REST

  • Resource (Recurso): Objeto que representa datos (usuario, producto, etc.)
  • Endpoint: URL donde accedes a un recurso
  • HTTP Methods: GET, POST, PUT, DELETE, PATCH
  • Status Codes: Códigos que indican resultado de la operación
  • JSON: Formato de intercambio de datos

📡 Métodos HTTP

  • GET: Obtener datos (no modifica)
  • POST: Crear nuevo recurso
  • PUT: Actualizar recurso completo
  • PATCH: Actualizar recurso parcial
  • DELETE: Eliminar recurso
  • HEAD: Como GET pero sin body

🎯 Status Codes HTTP

  • 2xx (Éxito)
    • 200: OK - Éxito general
    • 201: Created - Recurso creado
    • 204: No Content - Sin contenido que retornar
  • 3xx (Redirección)
    • 301: Moved Permanently
    • 304: Not Modified
  • 4xx (Error del cliente)
    • 400: Bad Request - Solicitud inválida
    • 401: Unauthorized - No autenticado
    • 403: Forbidden - Sin permisos
    • 404: Not Found - Recurso no existe
  • 5xx (Error del servidor)
    • 500: Internal Server Error
    • 503: Service Unavailable

✅ Validación de Datos

# Validar en servidor
app.post('/usuarios', (req, res) => {
    const { nombre, email, edad } = req.body;
    
    // Validar campos
    if (!nombre || nombre.length < 3) {
        return res.status(400).json({
            error: "Nombre inválido"
        });
    }
    
    if (!email || !email.includes('@')) {
        return res.status(400).json({
            error: "Email inválido"
        });
    }
    
    if (edad < 0 || edad > 150) {
        return res.status(400).json({
            error: "Edad inválida"
        });
    }
    
    // Si todo es válido, procesar
    res.status(201).json({ id: 1, nombre, email, edad });
});

📚 Ejemplo Completo - API de Tareas

# app.py (Flask)
from flask import Flask, request, jsonify

app = Flask(__name__)

tareas = []

@app.route('/api/v1/tareas', methods=['GET'])
def obtener_tareas():
    return jsonify(tareas)

@app.route('/api/v1/tareas', methods=['POST'])
def crear_tarea():
    data = request.json
    if not data or 'titulo' not in data:
        return jsonify({'error': 'título requerido'}), 400
    
    tarea = {
        'id': len(tareas) + 1,
        'titulo': data['titulo'],
        'completada': False
    }
    tareas.append(tarea)
    return jsonify(tarea), 201

@app.route('/api/v1/tareas/', methods=['GET'])
def obtener_tarea(id):
    for tarea in tareas:
        if tarea['id'] == id:
            return jsonify(tarea)
    return jsonify({'error': 'no encontrada'}), 404

@app.route('/api/v1/tareas/', methods=['DELETE'])
def eliminar_tarea(id):
    global tareas
    tareas = [t for t in tareas if t['id'] != id]
    return '', 204
