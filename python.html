🐍 Python - Lenguaje de Programación

Python es uno de los lenguajes de programación más populares y versátiles. Sua sintaxis clara y fácil de aprender lo hace ideal para principiantes, pero es lo suficientemente poderoso para aplicaciones complejas. Se usa en web, ciencia de datos, automatización y muchas más áreas.

🎯 ¿Por Qué Python?

  • Fácil de aprender: Sintaxis clara y legible
  • Versátil: Web, datos, automatización, IA
  • Gran comunidad: Miles de librerías disponibles
  • Demandado: Altamente buscado en el mercado laboral
  • Productivo: Escribe menos, haz más

📝 Sintaxis Básica

Variables y Tipos de Datos

# Variables
nombre = "Juan"
edad = 28
altura = 1.75
activo = True

# Tipos de datos
tipo(nombre)      # str
tipo(edad)        # int
tipo(altura)      # float
tipo(activo)      # bool

# Print
print("Hola, mundo!")
print(f"Me llamo {nombre} y tengo {edad} años")

Operadores

# Aritméticos
suma = 10 + 5      # 15
resta = 10 - 5     # 5
mult = 10 * 5      # 50
div = 10 / 5       # 2.0
mod = 10 % 3       # 1
pot = 2 ** 3       # 8

# Comparación
10 > 5             # True
10 == 5            # False
10 != 5            # True

# Lógicos
True and False     # False
True or False      # True
not True           # False

Estructuras de Control

# If - Elif - Else
edad = 18
if edad >= 18:
    print("Eres mayor de edad")
elif edad >= 13:
    print("Eres adolescente")
else:
    print("Eres niño")

# Bucle while
contador = 0
while contador < 5:
    print(contador)
    contador += 1

# Bucle for
for i in range(5):
    print(i)

for nombre in ["Juan", "María", "Carlos"]:
    print(f"Hola {nombre}")

🔧 Funciones

# Función básica
def saludo():
    print("¡Hola!")

saludo()

# Función con parámetros
def saludar(nombre):
    print(f"¡Hola, {nombre}!")

saludar("Juan")

# Función con retorno
def sumar(a, b):
    return a + b

resultado = sumar(5, 3)
print(resultado)    # 8

# Parámetros por defecto
def presentarse(nombre, edad=0):
    print(f"Me llamo {nombre} y tengo {edad} años")

presentarse("Juan", 28)
presentarse("María")

📊 Estructuras de Datos

Listas

frutas = ["manzana", "plátano", "naranja"]
numeros = [1, 2, 3, 4, 5]

# Acceso
print(frutas[0])        # manzana
print(frutas[-1])       # naranja

# Métodos
frutas.append("fresa")  # Añadir
frutas.remove("plátano") # Eliminar
len(frutas)             # Longitud
frutas.sort()           # Ordenar

Diccionarios

persona = {
    "nombre": "Juan",
    "edad": 28,
    "ciudad": "Madrid"
}

# Acceso
print(persona["nombre"])    # Juan
print(persona.get("edad"))  # 28

# Modificar
persona["edad"] = 29
persona["profesión"] = "Developer"

# Recorrer
for clave, valor in persona.items():
    print(f"{clave}: {valor}")

Tuplas

# Tuplas (inmutables)
coordenadas = (10, 20)
print(coordenadas[0])    # 10

# Desempacar
x, y = coordenadas
print(x, y)              # 10 20
